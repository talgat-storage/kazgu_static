ИЗЛУЧЕНИЕ АТОМА ВОДОРОДА

Разреженные газы можно возбудить так, чтобы они испускали свет. Этого можно достичь сильным нагреванием или приложением к газоразрядной трубке высокого напряжения. В начале XIX века было обнаружено, что спектр излучения разреженных газов является линейчатым, т.е. интенсивность излучения отлична от нуля только для некоторых длин волн. При этом набор длин волн излучения свой для каждого вида атомов.

Водород представляет собой простейший атом, так как состоит из ядра и одного электрона. У водорода самый простой спектр. Расстояния между линиями в видимой части спектра уменьшаются в соответствии с законом, найденным в 1885 г. швейцарским школьным учителем и одновременно доктором Базельского университета Иоганном Бальмером,

    1              1       1
          ―  = R (  ― -  ― )  ,
      λ              22     m2

	  где m = 3, 4, 5,... ,
	   R=1.097·107 м-1 ( постоянная Ридберга).

Позднее было обнаружено, что в ультрафиолетовой и инфракрасной областях имеются другие серии линий, длины волн которых описываются подобной формулой. Для объяснения линейчатого характера спектра излучения. Н.Бор ввел понятие о стационарных состояниях атома с дискретным набором энергий. Те же закономерности в спектрах наблюдаются для водородоподобных атомов. При сообщении атому большой энергии может произойти его ионизация, т.е. отрыв электрона от атома.

Общая формула для длин волн линий серии атома водорода может быть записана в виде

    1              1       1
          ―  = R (  ― -  ― )  ,
      λ              n2     m2

 где  n = 1, 2, 3,..., m = n+i, i = 1, 2,....
 Серия Лаймана: n = 1, m = 2, 3, 4,...,
серия Бальмера: n = 2, m = 3, 4, 5, ...,
серия Пашена: n = 3, m = 4, 5, 6,... и т.д.

Н.Бор предположил, что испускание света происходит, когда электрон переходит из состояния с большей энергией в состояние с меньшей энергией, причем энергии состояний дискретны. При каждом переходе испускается фотон, энергия которого определяется соотношением ( закон сохранения энергии! )

hν = E(m) - E(n),

где m - номер состояния с большей энергией или, как говорят, более высокого энергетического уровня, n - номер более низкого энергетического уровня.

Если энергию кванта выражать в электронвольтах, то

          1         1
 hν = 13.6 (  ―  -   ― )  эВ.
           n2       m2

Энергия взаимодействия двух зарядов равна нулю, если они удалены на бесконечное расстояние. Если для сближения зарядов требуются затраты энергии, то потенциальная энергия (энергия взаимодействия) считается положительной. Электронам, связанным в атоме, отвечает значение энергии   E < 0, т.к. они притягиваются к ядру:

  13.6
    E = - ――    эВ,
   n2
 n = 1, 2, 3, ...

Спектры водородоподобных ионов, состоящих из ядра и одного электрона (He+, Li++, ...), имеют характер, аналогичный спектру водорода, но соответствующие частоты в Z2 раз превышающие частоты в спектре водорода
( Z - число протонов в ядре ):

               1         1
hν = 13.6·Z2 (   ―   -   ―  ) эВ,
                n2       m2
где n = 1, 2, 3,...,
 m = n+i,  i = 1, 2, ...

Минимальную энергию, необходимую для вырывания электрона из атома, называют энергией связи электрона или энергией ионизации Ei. Если энергия поглощенного кванта больше энергии связи электрона, то разность их сообщается электрону в виде кинетической энергии (действует закон сохранения энергии !)

mv2/2 = hν - Ei
